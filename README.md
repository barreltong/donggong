# 동공 (Donggong)

히토미 전용 리더 앱입니다.

## 주요 기능

### 특수 기능
- **자체 우회 적용**: 자체적으로 검열을 우회하기 위한 모듈을 삽입하여 cdn 이외 `hitomi.la`에 접근할 일이 생기는 경우 이를 자동 우회

### 탐색 & 검색
- **무한 스크롤 / 페이지 뷰** 선택 가능
- **태그 검색** 지원 (`artist:`, `female:`, `male:`, `group:`, `character:` 등)
- **언어 필터** (한국어 / 영어 / 일본어 / 중국어 / 전체)

### 리더
- **세로 페이지**: 한 장씩 위아래로 넘김
- **가로 페이지**: 한 장씩 좌우로 넘김
- **웹툰식 보기**: 세로로 쭉 내리면서 보는 방식

### 즐겨찾기
- **JSON 백업/복구**: 폰 바꿔도 데이터 안 날아감

### 테마
- 라이트 / 다크 / OLED 다크 지원
- Material Design 3 기반

## 설정 항목

| 설정 | 설명 |
|------|------|
| 앱 언어 | 한국어 / 영어 |
| 콘텐츠 언어 | 검색 시 기본 필터 언어 |
| 리스팅 모드 | 무한 스크롤 vs 페이지 뷰 |
| 카드 스타일 | 썸네일 위주 vs 상세정보 위주 |
| 리더 모드 | 세로 페이지 / 가로 페이지 / 웹툰식 보기 |

## 기술 스택

- **프레임워크**: Flutter
- **DB**: SQLite (로컬 저장)
- **이미지 캐싱**: cached_network_image
